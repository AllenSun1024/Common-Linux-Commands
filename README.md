# Common-Linux-Commands🤔
This repository is used to record common shell commands in Linux.

## wget🐬
> wget用于下载来自于网络的文件，支持HTTP、HTTPS和FTP协议。  

```
wget [-c/-r/...] http://www.linuxsense.org/xxxx/xxx.tar.gz
```

| 参数 | 功能                 |
| :--- | :--------            |
| -c   | 断点续传             |
| -r   | 递归下载             |
| -b   | 后台下载             |
| -O   | 将文件下载到指定目录 |
